对于三种背包问题：01，完全，多重
他们之间的混合问题如何解决呢？
比如，有些物品只能拿一个，有些物品只能拿有限个，有些物品能拿无限个。
事实上，多重背包可以转换成01背包，这一点在多重背包问题里已经解释过了。
所以，混合问题其实就是01背包和完全背包的混合。

我们假设题目给出三个列表：
bool b[i] 若值为True，则表示第i个物品可以拿无限个。
v[i]和w[i]和以前一样。
接下来是主逻辑：
for i in range(1,len(b)):#假设我们第一行经过了初始化，所以i从1开始。
	if b[i]:
		for c in range(C+1):
			if c>=w[i]:
				memo[c]=max(memo[c],v[i]+memo[c-w[i]])
	else:
		for c in range(C,-1,-1):
			if c>=w[i]:
				memo[c]=max(memo[c],v[i]+memo[c-w[i]])

注意在拆分多重背包的时候，向列表b中添加对应的False信息。
